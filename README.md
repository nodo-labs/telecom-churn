# Telecom Churn Prediction

Proyecto final de Data Science enfocado en predecir la cancelación de clientes
(churn) para la empresa de telecomunicaciones Interconnect.

## Objetivo
Desarrollar un modelo de machine learning que estime la probabilidad de churn
de clientes, con el fin de apoyar estrategias de retención proactivas.

## Datos
Los datos utilizados en este proyecto son proporcionados por TripleTen y no se
incluyen en este repositorio.

Archivos esperados:
- contract.csv
- personal.csv
- internet.csv
- phone.csv

Ruta esperada:
- /datasets/final_provider/

## Métricas
- Métrica principal: AUC-ROC
- Métrica adicional: Accuracy

## Estructura del repositorio
- telecom_churn.ipynb: desarrollo completo del proyecto
- environment.yml: definición del environment reproducible
